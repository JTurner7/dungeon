# Dungeon

The beginnings of a homelab. The name is because I love dungeons and dragons and hope to theme my homelab and devices around it.

**Warning:** This only works on my WSL environment. Not to be deployed to general environments yet.

## Apps

- [Homarr](https://homarr.dev/) for a homepage
- [Mealie](https://docs.mealie.io/) for a recipe manager
- [Searxng](https://docs.searxng.org/) for a search engine
- [kube-prometheus-stack](https://github.com/prometheus-community/helm-charts/tree/main/charts/kube-prometheus-stack) for observability
  - [Grafana](https://grafana.com/docs/) for dasboarding and graphing
  - [Prometheus](https://prometheus.io/docs/introduction/overview/) for metric management
